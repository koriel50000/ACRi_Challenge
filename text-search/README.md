# 問題

テキストからパターンと一致する部分を見つけ、その位置を出力してください。

## 入力

* `in_text`
  * 入力テキスト
  * 16,384 文字
  * `a` ~ `z` の小文字アルファベットのみ含みます
* `in_pat`
  * 検索パターン
  * 32 文字
  * `a` ~ `z` の小文字アルファベットおよび `?` のみ含みます
  * `?` は任意の文字と一致します

## 出力

* `out_match_pos`
  * パターンと一致した位置を先頭から順に最大 16 箇所まで格納してください
* `out_match_num`
  * パターンと一致した箇所の数を格納してください
